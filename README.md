# 할일 관리 (To-Do Manager)

React 기반의 직관적이고 모던한 할일 관리 웹 애플리케이션입니다.

## 주요 기능

### 📅 날짜/시간 관리
- 날짜와 시간을 지정하여 할 일 추가
- 오늘, 이번주, 나중에 카테고리로 자동 분류

### 📁 폴더 시스템
- 업무, 개인, 공부 등 폴더별 할일 분류
- 폴더별 파스텔 컬러 지정
- 커스텀 폴더 추가/삭제 가능

### 📊 대시보드
- 오늘의 일정 한눈에 보기
- 완료율 실시간 표시
- 긴급 일정 우선 표시

### 💾 로컬 스토리지
- 브라우저 새로고침해도 데이터 유지
- 별도 서버 없이 로컬에서 동작

### 📱 반응형 디자인
- 데스크톱, 태블릿, 모바일 모두 지원
- 터치 친화적 UI

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:5173 을 열어주세요.

### 빌드

```bash
npm run build
```

## 기술 스택

- **React** - UI 라이브러리
- **Vite** - 빌드 도구
- **CSS3** - 스타일링 (애니메이션 포함)
- **Local Storage** - 데이터 저장

## 프로젝트 구조

```
src/
├── components/          # React 컴포넌트
│   ├── Dashboard.jsx    # 오늘의 일정 대시보드
│   ├── CategoryTabs.jsx # 카테고리 탭
│   ├── FolderManager.jsx # 폴더 관리
│   ├── TodoForm.jsx     # 할일 추가 폼
│   ├── TodoItem.jsx     # 할일 항목
│   └── TodoList.jsx     # 할일 목록
├── hooks/              # 커스텀 훅
│   └── useTodos.js     # 할일 관리 로직
├── utils/              # 유틸리티 함수
│   ├── dateUtils.js    # 날짜 처리
│   └── storage.js      # 로컬스토리지 관리
├── App.jsx             # 메인 앱 컴포넌트
├── App.css             # 스타일
└── main.jsx            # 엔트리 포인트
```

## 사용 방법

1. **할일 추가**: 상단 폼에서 제목, 메모, 날짜, 시간, 폴더를 선택하고 추가
2. **카테고리 보기**: 오늘/이번주/나중에 탭으로 전환
3. **폴더 필터**: 왼쪽 사이드바에서 폴더 선택
4. **완료 표시**: 체크박스 클릭으로 완료/미완료 토글
5. **삭제**: 각 항목의 X 버튼으로 삭제
